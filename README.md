# 2FA 验证码生成器

一个基于 Python 和 Tkinter 的本地 2FA 动态验证码生成工具。

## 功能特点
- 支持 16位 / 32位 等标准 Base32 2FA 密钥解析
- 实时显示 30秒 倒计时，并支持一键复制到剪贴板
- **[新增]** 支持从本地图片直接读取并解析 2FA 二维码
- **[新增]** 支持一键识别系统剪贴板图片，或自动全屏扫描二维码（无需保存图片）
- 本地记忆密钥功能（加密/明文存储于本地，不随代码上传）

## 下载免安装版
Windows 用户可以直接前往右侧的 **[Releases]** 页面下载打包好的 `.exe` 文件，双击即可直接运行，无需安装任何环境。

## 源码使用方法（面向开发者）
1. 安装依赖：`pip install -r requirements.txt`
2. 运行程序：`python main.py`
